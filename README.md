 # 📱 Mobile Price Range Classifier

A comparative study of supervised ML classification algorithms to predict 
mobile phone price ranges based on hardware specifications.

## Algorithms Compared
- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- Random Forest
- SVM (Linear Kernel)

## Dataset
[Mobile Price Classification — Kaggle](https://www.kaggle.com/datasets/iabhishekofficial/mobile-price-classification)

## Tools Used
Python | Jupyter Notebook | Scikit-learn | Pandas | Seaborn
